# Rigid Body Dynamics 2D

**日期**: 2026-04-10  
**系列**: 每日编程实践 #55  
**难度**: ⭐⭐⭐⭐

## 项目简介

实现了一个完整的 2D 刚体动力学模拟器，包含：

- **SAT 分离轴定理** 碰撞检测（矩形 vs 矩形、圆 vs 圆、圆 vs 矩形）
- **脉冲解算（Impulse-based）** 碰撞响应
- **摩擦力** 模拟（静摩擦 + 动摩擦）
- **Baumgarte 位置修正** 防止穿透漂移
- **半隐式 Euler 积分**
- **软光栅化** 可视化输出

## 场景描述

场景包含 14 个刚体：
- 3 个静态边界（地面 + 左右墙）
- 2 个静态斜台（斜坡 + 平台）
- 4 个动态矩形（层叠箱子，带初速度）
- 4 个动态圆形（带不同初速度方向）
- 1 个重型矩形（质量 5kg，带角速度）

## 技术实现

### SAT 碰撞检测
```
对每条边的法线轴进行投影
→ 找到所有轴上的最小重叠深度
→ 若任意轴无重叠 → 分离，无碰撞
→ 最小重叠轴 = 碰撞法线
```

### 脉冲解算
```
相对速度 vRel = vA - vB（在接触点）
法向脉冲: j = -(1+e) * vRel·n / (invMassSum + 转动贡献)
切向脉冲: Coulomb 摩擦 (静摩擦 ≤ μ * j)
```

### Baumgarte 修正
```
每步修正穿透：correction = max(depth - slop, 0) * 0.4 / invMassSum
位置 += correction * invMass * normal
```

## 物理参数

- 重力：600 px/s²（向下）
- 时间步：1/60s（60 FPS）
- 碰撞解算迭代：10 次/帧
- 恢复系数 e：0.3-0.4
- 摩擦系数 μ：0.3-0.4
- 角阻尼：0.98^(dt*60)

## 学习收获

1. **SAT 定理** 对凸多边形碰撞检测的优雅实现
2. **脉冲解算** 比惩罚力法更稳定，无刚度参数调节问题
3. **Baumgarte 修正** 是防止穿透漂移的关键
4. **圆 vs 矩形** 需要将圆心变换到矩形局部空间再做 clamp
5. **旋转惯量** I = (1/12)m(w²+h²) 影响角速度变化
