# Procedural Terrain Renderer

> 每日编程实践 Day 48 · 2026-04-06

## 项目简介

使用 C++ 从零实现程序化地形渲染器，集成 Perlin 噪声高度图生成、fBm 分形布朗运动、法线计算、多生物群系着色、Phong 光照模型、大气雾效和软光栅化投影。

## 编译运行

```bash
g++ main.cpp -o output -std=c++17 -O2 -Wall -Wextra
./output
# 生成 terrain_output.png (800x600)
```

## 技术要点

- **Perlin 噪声 + fBm**：7层分形叠加生成自然高度图（persistence=0.55, lacunarity=2.0）
- **多生物群系着色**：根据高度和坡度自动分配雪顶/岩石/森林/草地/沙滩/海水
- **法线计算**：中心差分法从高度图计算精确法线
- **Phong 光照**：环境光 + 漫反射 + 镜面反射
- **大气雾**：指数衰减雾效增加景深感
- **软光栅化**：基于透视投影的三角形扫描线填充，带深度缓冲
- **验证通过**：像素均值 178.1，标准差 46.9，文件 251KB

## 渲染参数

- 分辨率：800×600
- 地形网格：200×200（step=2）
- 地形尺寸：12×12 世界单位，高度缩放 3.5
- 相机：(-1, 5.5, -7.5) → (0.5, 1.0, 3.0)，FOV=65°
