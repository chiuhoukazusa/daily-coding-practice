# Bezier 曲线绘制器

## 项目描述
实现 Bezier 曲线的绘制，支持2阶、3阶、4阶曲线，使用 De Casteljau 递归算法。
生成多个示例曲线，展示不同控制点配置下的效果。

## 核心技术
- **De Casteljau 算法**: 递归线性插值
- **参数方程**: t ∈ [0, 1] 参数化曲线
- **控制点系统**: 通过控制点定义曲线形状
- **平滑采样**: 自适应步长或均匀采样

## 编译运行
```bash
g++ -std=c++17 -O2 bezier.cpp -o bezier
./bezier
```

输出文件：
- `bezier_quadratic.png` - 二阶曲线
- `bezier_cubic.png` - 三阶曲线
- `bezier_quartic.png` - 四阶曲线
- `bezier_combined.png` - 组合展示

## 数学原理

### De Casteljau 算法
对于 n 个控制点 P₀, P₁, ..., Pₙ₋₁，递归定义：

```
B(t) = (1-t) * B₀₋ₙ₋₂(t) + t * B₁₋ₙ₋₁(t)
```

基础情况：`B(t) = Pᵢ`

### 二阶 Bezier（3个控制点）
```
B(t) = (1-t)² * P₀ + 2(1-t)t * P₁ + t² * P₂
```

### 三阶 Bezier（4个控制点）
```
B(t) = (1-t)³ * P₀ + 3(1-t)²t * P₁ + 3(1-t)t² * P₂ + t³ * P₃
```

## 预期效果
- 平滑曲线通过控制点附近
- 起点和终点精确匹配首尾控制点
- 曲线形状由中间控制点拉伸

## 技术挑战
1. 精确的递归实现
2. 合适的采样密度（避免锯齿/性能问题）
3. 视觉化控制点和曲线的关系
4. PNG 输出和抗锯齿线条
