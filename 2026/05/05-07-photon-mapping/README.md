# Photon Mapping Renderer

经典两阶段全局光照算法实现，在 Cornell Box 场景中展示间接光照、镜面反射和玻璃焦散效果。

## 编译运行

```bash
g++ main.cpp -o photon_map -std=c++17 -O2
./photon_map
```

输出：`photon_map_output.png`（512×512，约446KB）

## 算法原理

### Phase 1：光子发射
- 从面光源（球形近似）随机发射光子
- 光子携带 RGB 功率，通过 Path Tracing 风格多次弹射
- 漫反射面存入**全局光子图**（间接光照）
- 经镜面/折射后打到漫反射面存入**焦散光子图**（Caustics）
- 俄罗斯轮盘决定是否继续追踪（最大深度 8）

### Phase 2：渲染（Radiance Estimation）
- 从摄像机出发，路径追踪到漫反射面
- 直接光照：Next Event Estimation，对光源采样
- 间接光照：查询**全局光子图**，用 Epanechnikov 核估计辐射度
- 焦散：查询**焦散光子图**，小半径高精度估计
- 镜面/折射：递归路径追踪

### kd-tree 加速
- 中位数切割构建（选最大范围轴）
- 最近邻球形查询，O(log N) 复杂度

## 场景构成

| 物体 | 材质 | 特性 |
|------|------|------|
| Cornell Box 五面墙 | 漫反射（白/红/绿） | 间接光照 |
| 左球 | 完美镜面 | 镜像反射 |
| 右球 | 玻璃（IOR=1.5）| 折射+焦散 |
| 小球 | 黄色漫反射 | 间接光照着色 |
| 顶部球 | 面光源（15,15,15）| 光子发射源 |

## 技术要点
- 光子映射两阶段算法（Jensen 1996）
- kd-tree 最近邻加速（中位数切割）
- 全局光子图 + 焦散光子图分离存储
- Schlick Fresnel 近似
- ACES 色调映射 + sRGB Gamma 校正
- 200,000 光子发射，32 spp 渲染
