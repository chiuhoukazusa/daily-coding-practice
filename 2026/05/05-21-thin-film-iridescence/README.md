## 技术要点
- **薄膜干涉物理**: 光在薄膜两个界面的反射光相互叠加产生干涉
- **菲涅尔方程**: 精确计算 s/p 偏振的振幅反射/透射系数
- **波动光学**: 相位差 δ = 4π·n₂·d·cos(θ₂)/λ，波长依赖性产生色散
- **CIE XYZ 色匹配函数**: 将干涉光谱积分转换为可感知的 RGB 颜色
- **多种薄膜材质**: 肥皂泡 (n=1.33)、油膜 (n=1.47)、甲虫翅膀 (n=1.50)
- **厚度渐变**: 不同材质有不同的厚度变化模式（重力、湍流、均匀）
