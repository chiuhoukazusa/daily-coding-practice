# Light Field Renderer - 光场渲染器

基于双平面参数化（Two-Plane Parameterization）的 4D 光场渲染器，采用 Lumigraph 思路捕捉并重建任意视角图像。

## 编译运行

```bash
g++ main.cpp -o output -std=c++17 -O2
./output
```

## 技术要点

- **双平面参数化 (u,v,s,t)**：UV 平面存放相机阵列，ST 平面描述每个相机的像素坐标
- **5×5 相机网格采集**：25 个视点，每个 128×128 像素，共追踪 409,600 条光线
- **双线性插值视角合成**：通过四邻域相机图像插值重建任意中间视点
- **软光栅化场景**：5 个球体（金属/漫反射/透明）+ 2 个平面 + 2 个光源
- **Blinn-Phong 着色**：漫反射 + 镜面反射 + 环境光 + 阴影射线
- **反射渲染**：金属球体支持最多 2 次递归反射
- **极简 PNG 写出**：无外部依赖，手写 deflate-store + zlib + CRC32

## 渲染参数

| 参数 | 值 |
|------|-----|
| UV 网格尺寸 | 5×5 |
| ST 分辨率 | 128×128 |
| 相机间距 | 0.25 units |
| 输出分辨率 | 900×900 |
| 渲染时间 | ~0.11s |
